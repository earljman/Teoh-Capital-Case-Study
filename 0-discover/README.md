## Synthesis for Phase 1 scoping

**White space:** AI-orchestrated warehouse execution at mid-market speed/price — between transactional tools (CartonCloud, Cin7, WiseWMS) and enterprise AI (Manhattan, Easy WMS automation stack).

**Candidate top 3 Phase 1 features** (from competitive gaps + journey opportunities):

1. **AI Task Orchestration** — dynamic prioritization and pick-path / task interleaving  
2. **AI Exception Copilot** — ranked exceptions with recommended actions and override  
3. **AI Ops Command Center** — shift brief, live risk, and accuracy signals for supervisors  

**Market anchor:** Global WMS ~$4.0–4.8B (2026); mid-market cloud SAM ~$1.0–1.5B; Y3 SOM planning band ~$15–45M ARR (assumption-heavy).
